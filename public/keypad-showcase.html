pb-numpad

The PB_Numpad component displays a numeric pad of two kinds western numeral and arabic digits depending of the selected lang or the browser default through the i18n lib.

Available attributes:

  shuffled: string. If "true" the digits do not appear in an ordered fashion. Default: "false".

  fixedcodelength: string. If "false" the code length is not of a fixed length and a submit button appears at the bottom of the numpad. Default: "true".

  codelength: string. If "fixedcodelength" attribute is true, determines the code length at which the numpad will automatically executes the callback function. Default: "6"

  callbackfunc: the callback function that will be executed by the numpad.

Default setup

'pb-numpad shuffled="true"'

'pb-numpad codelength="4" '

'pb-numpad fixedcodelength="false" '
